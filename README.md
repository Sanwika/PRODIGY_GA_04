# PRODIGY_GA_04

## Image-to-Image Translation with cGAN (pix2pix)

### Objective
Implement an image-to-image translation model using a Conditional Generative Adversarial Network (cGAN) called pix2pix.

### Tools & Technologies
- Python
- Google Colab
- PyTorch
- pix2pix
- GAN

### Dataset
Facades Dataset

### Results
Successfully translated facade label images into realistic building images using a pre-trained pix2pix model.

### Learning Outcomes
- Understood the concept of Conditional GANs.
- Learned how pix2pix performs image-to-image translation.
- Gained hands-on experience with pre-trained deep learning models.
